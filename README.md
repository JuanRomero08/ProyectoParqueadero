# Proyecto Parqueadero

## Descripción del proyecto
Este proyecto consiste en el desarrollo de un sistema de gestión de parqueadero realizado en Java.  
El sistema permite llevar el control básico del parqueadero, como el registro de vehículos, el manejo de ingresos y salidas, y la visualización de la información a través de una interfaz gráfica.

## Herramientas y tecnologias utilizadas
- Java JDK 21  
- Apache Maven  
- NetBeans IDE  

## Instrucciones para ejecutar el proyecto

### Requisitos previos
- Tener instalado Java JDK 21
- Tener instalado NetBeans IDE
- Tener Git instalado (opcional, si se clona el repositorio)

### Pasos para ejecutar

1. Clonar el repositorio desde GitHub:
   git clone https://github.com/JuanRomero08/ProyectoParqueadero.git
   
2. Abrir el proyecto en NetBeans:
-Abrir NetBeans
-Ir a File → Open Project
-Seleccionar la carpeta ProyectoParqueadero

3. Ejecutar el proyecto:
-Click derecho sobre el proyecto
-Seleccionar Run
